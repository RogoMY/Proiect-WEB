ReFi: Cautatorul de Resurse pentru Programare Creativa - Raport Scholarly HTML

Autori
Luca-Matei Cozloschi, Mihai Rogojina

Abstract

Acest document detaliaza aspectele tehnice si interactiunea cu utilizator ale proiectului ReFi (Resource Finder), o platforma pentru gestionarea si descoperirea resurselor legate de programarea creativa.

Introducere

Scop

Platforma ReFi este conceputa sa imbunatateasca modul in care programatorii acceseaza si gestioneaza resursele de programare pentru diverse discipline creative, folosind materialele din acest repository GitHub. Ofera un mediu clar si eficient, care faciliteaza gasirea si utilizarea resurselor cum ar fi biblioteci, unelte si tutoriale specifice programarii creative.

Audienta tinta

Utilizatorii principali ai ReFi includ programatorii, educatorii si studentii interesati de programarea creativa si artele digitale.

Viziune

ReFi este un produs autonom conceput pentru a facilita gestionarea si descoperirea resurselor de programare creativa. Este, de asemenea, proiectat sa fie scalabil, cu potentialul de a deveni membru al unei familii de produse mai mari pentru dezvoltarea software si managementul resurselor digitale.

In calitate de componenta a unui sistem mai mare, ReFi va putea interactiona cu alte sisteme externe, cum ar fi baze de date de resurse educationale si platforme de colaborare pentru dezvoltatori. Acesta va servi drept intermediar intre utilizatorii care cauta resurse de programare creativa si diversele repozitorii pe care le ofera.

Descriere generala

Platforma integreaza diverse functionalitati pentru a simplifica procesul de gestionare si descoperire a resurselor.

Perspectiva Produsului

Aplicatia isi propune sa ofere o modalitate rapida de gasire a resurselor pentru programarea creativa, fiind nevoie de o simpla descriere(limbajul de programare,maniera de interactiune, rezultatul dorit,etc.) pentru a obtine materiale relevante.

Functiile produsului

  • Cautare si filtrare a resurselor bazate pe informatiile oferite in limbaj natural de catre utilizator.
  • Sistem de autentificare si inregistrare al utilizatorului
  • Sistem de afisare al istoricului cautarilor
  • Sistem de salvare al materialelor marcate ca fiind "favorite".
  • ADMIN Dashboard: Gestionarea conturilor de utilizatori.
  • ADMIN Dashboard: Aplicarea operatiilor CRUD asupra materialelor disponibile utilizatorilor.

Mediu de operare

Aceasta aplicatie web este optimizata pentru accesibilitate universala, fiind compatibila cu o gama larga de dispozitive, incluzand computere de tip desktop, laptopuri, tablete si telefoane mobile. Indiferent de sistemul de operare — fie ca este vorba de Android, iOS, Windows sau Linux — utilizatorii pot accesa aplicatia eficient si fara probleme direct prin intermediul browserului lor preferat. Aceasta flexibilitate asigura o experienta de utilizare fluida si accesibila de oriunde.

Constrangeri

Aplicatia nu include un sistem de comunicare intre utilizatori, ceea ce impiedica partajarea de resurse sau oferirea de feedback pentru materialele existente. De asemenea, materialele nu pot fi descarcate direct din aplicatia web ReFi; utilizatorii trebuie sa gestioneze singuri continutul accesat prin link-urile disponibile. Aceasta configuratie necesita o abordare mai independenta din partea utilizatorilor pentru a naviga si utiliza resursele.

Ghid al utilizatorului

Pagina de home are rolul de a explica, clar si concis, cum se poate utiliza aplicatia web ReFi, în modul în care a fost intenționat.

Interfata grafica

Descriere interfata si motivare

Aplicatia ReFi este echipata cu o interfata vizuala moderna si atractiva, proiectata sa imbunatateasca semnificativ experienta utilizatorului. Adaptabila si flexibila, aceasta asigura o navigare fluida si intuitiva atat pe dispozitive cu ecrane de dimensiuni mari, cat si pe cele mobile, cu ecrane mai mici. Designul responsiv al interfetei se adapteaza automat la diferitele dimensiuni ale ecranelor, garantând o utilizare eficienta si placuta, indiferent de platforma aleasa.

Dezvoltarea interfetei utilizator a fost realizata folosind HTML5, CSS3 si JavaScript, asigurând compatibilitatea cu cele mai recente standarde web si tehnologii. HTML5 a fost utilizat pentru structurarea semantica a continutului, în timp ce CSS3 a adaugat stiluri personalizate (pentru a oferi un mediu de navigare estetic utilizatorului) si adaptabilitate printr-o varietate de medii de interogare (media queries) si flexbox-uri (acestea ofera capacitatea aplicatiei de a-si pastra o structura organizata si usor de vizualizat pentru orice dimensiune a ecranului). JavaScript a imbogatit interactiunea utilizatorului cu elemente dinamice, precum declansarea meniului dedicat ecranelor mici pe baza unui buton sau afisarea rezultatelor pe baza inputului din bara de search.

Functionalitati

Autentificarea

Autentificarea utilizatorilor se efectueaza printr-o pagina special destinata acestui scop, unde este necesar sa se completeze informatiile solicitate (username si parola). Dupa introducerea datelor, utilizatorii vor apasa pe butonul „login”. Daca informatiile sunt valide, vor fi redirectionati catre pagina principala sau catre admin dashboard (in cazul in care logarea a fost efectuata pentru un cont de administrator). Este important de mentionat ca pentru autentificare este necesar sa existe un cont de utilizator preexistent.

Inregistrarea

Procedura de inregistrare incepe cu completarea unui formular, unde utilizatorul introduce email-ul, parola si confirmarea parolei. Dupa apasarea butonului „register” si verificarea corectitudinii informatiilor (email-ul trebuie sa fie intr-un format acceptat si parolele trebuie sa corespunda), utilizatorul este redirectionat catre pagina de login.

Sistem de afisare al istoricului cautarilor

Pe pagina dedicata vizualizarii istoricului, utilizatorul isi poate vedea si reaccesa ultimele sale cautari. De asemenea, este afisata data si ora cautarii.

Sistem de afisare al materialelor favorite

Pe pagina dedicata vizualizarii materialelor favorite, utilizatorul va putea accesa printr-un singur click materialele pe care le-a marcat ca fiind "favorite".

Admin Dashboard

  • Gestionarea utilizatorilor:

    Administratorul poate sterge un utilizator din baza de date, sau ii poate actualiza informatiile (adresa de e-mail, parola). De asemenea, pentru a gasi mai usor un utilizator, se poate folosi de functia de search pe baza username-ului sau al e-mail-ului.

  • Gestionarea materialelor:

    Administratorul, prin apasarea butoanelor aferente, poate accesa formularele de adaugare de noi categorii (poate alege titlul si categoria-parinte, daca este necesar) sau de adaugare de noi elemente intr-o categorie deja existenta. La apasarea butonului de adaugare, valabil pe ambele formulare, se face o validare a informatiilor (ex. asocierea unui link valid unui material sau selectarea unei categorii), urmand sa fie afisate mesajele de eroare aferente, sau inchiderea formularului in caz de succes.

    Totodata, administratorul poate apasa pe butonul asociat unei categorii pentru a vizualiza, edita si sterge materialele cuprinse. De asemenea, administratorul poate sterge o categorie intreaga prin apasarea butonului de "delete" din dreapta titlului categoriei.
